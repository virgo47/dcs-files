Pictures taken from e-shop module pictures, cut to square and resized to 300x300.

Grey versions created in FastStone:
1. Colors -> Grayscale (Ctrl+G)
2. Colors -> Adjust Levels (Ctrl+L), Output Levels set to 64 and 192.
3. Save (Ctrl+S), it will ask for a name, don't worry.
