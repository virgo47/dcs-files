DCS World

  • NEW: Added support for Nvidia DLSS. Using upscaling, Deep Learning Super Sampling (DLSS) can dramatically improve game performance if you are Graphics Processor Unt (GPU) limited. Please note that if your CPU is limited, DLSS will have little to no performance effect. Owners of Nvidia 40-series graphics card will most benefit from this. With DLSS selected as the Upscaling option, you can further refine quality versus performance with Quality, Balance, Performance, and Ultra Performance presets. WIP
  • NEW: Added support for AMD FSR 2, FidelityFX Super Resolution (FSR) also using an upscaling algorithm to improve performance. Also like DLSS, it will have little to no effect if you are CPU-limited. WIP.
  • NEW: Anti-Aliasing. If Upscaling is set to OFF, three anti-aliasing modes are available.
    • Multi-Sampling Anti-Aliasing (MSAA)
    • Temporal Anti-Aliasing (TAA)
    • Deep Learning Anti-Aliasing (DLAA)
  • NEW: Added graphics option for Screen Space Shadows (SSS). This adds improved shadow detail to grass, bushes, and tree shadows. This is most evident with moving grass and tree shadows at long view distances. WIP.
  • NEW: Color Grading. Select between nine image color gradients that allow you to adjust the tint of the image to provide the desired effect. This can be most useful for video creators.
  • NEW: LOD Switch Factor. The Level of Detail (LOD) slider allows you to adjust the distance at which a unit transitions to lower level of detail models. 1 is the default value, but by setting a lower value, game performance can be increased.
  • NEW: Sharpening. The slider adjusts the image sharpness between 0.00 (default) and 1.00. Sharpening can help make it look crisp and clear by enhancing the edges of objects in the image. However, adding too much sharpness can actually make an image look worse, or it can lead to a loss in image detail. When using DLSS, we suggest a setting of 0.5 and 0.8-1.0 for FSR.
  • NEW: Cinematic Camera View - Right Control + F2. In combination with the mouse, the WASD keys can be used to move the camera forward, back, and laterally, and the Q and E keys can move the camera up and down. This allows the camera to be positioned in reference to the unit’s coordinate system, allowing much greater camera control for cinematic scenes.
  • NEW: 3D Models. Newly created models for the B-1B, B-52H, S-3B (S-3B Tanker to follow shortly), “Centurion” C-RAM, and SA-10 (S-300PS).
  • NEW: Ability to create, save, delete and rename custom weapons loadouts using in game rearm/refuel menu.

DCS Mirage F1 by Aerges

New features:

  • Added the new Mirage F1 BE version that includes the following major features:
  • Full multicrew capability.
  • Sight repeater.
  • Gunpods.
  • Priority control for the following systems: high-lift devices, airbrake, engine emergency regulation, green radio, nav aids, and radar.
  • Several systems repeaters.
  • Added AN/ALE-40 programmer clickability and specific view settings for the Mirage F1 single seater (check the flight manual for description).
  • The horizontal head movement limit is extended to an extra 10 cm when the canopy is open.
  • Implemented sight test procedure.
  • Added external CC-420 gun pods to all F1 variants.
  • Added axis for lights and intercom volume potentiometers.

Engine:

  • Adjusted engine flame-out with tail wind or reversed airflow. It may happen on the ground with extreme winds and at high altitudes with out-of-envelope manoeuvres that involve reversed airflow, such as a tailslide. The RPM has to be set high to minimise the phenomenon.
  • The engine starter button cover is spring loaded now (will close after a time delay).

Flight model:

  • Slightly increased high supersonic drag to better reflect acceleration data.

Autostart/Autostop:

  • Fixed auto start sequence. Now the throttle is checked/moved to idle in the cabin checks phase, to avoid a hot start chance.
  • Fixed auto start sequence. The Emergency gyromagnetic compass switch didn't set to On.
  • Fixed autostop sequence. The Emergency gyromagnetic compass switch didn't set to Off.

Manual and documentation:

  • Fixed kneeboard checklist, updated Flight Manual normal checklist and fixed auto-startup; regarding JPT emergency regulator switch, that must be in AUTO at cockpit preparation.
  • Updated flight manual to include Mirage F1 BE annex and other modifications.

Miscellaneous:

  • Corrected slat deployment and retraction routine to have the right slat deflection.
  • Now it is possible to select both radios for radio transmission on the Radio Selector Panel. Use keyboard or joystick buttons to press and hold more than one knob at once.
  • Fixed INS alignment error problems in certain conditions: when loading/unloading payloads, with wind/turbulence, MP...
  • Several 3D model and texture updates.
  • Fixed brake chute being self-deployed disregarding the handle position, after the aircraft repair is complete.
  • Fixed several single and quickstart missions (mainly due to missing pre-programmed radio channels or changed radio frequencies in South Atlantic terrain).
  • Selected radio for transmission (the radio background color) in Voice Chat will be displayed correctly now, when the aircraft was entered hot, while the server simulation has not started yet (server still paused).
  • Fixed sight camera position (it was not centred with the sight).

Known issues:
